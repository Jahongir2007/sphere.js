Getting started

Download .zip file in this site and copy downloaded folder in your app's folder and try it:

const sphere = require("./spherejs/sphere");
const app = sphere;
app.options({
    head: 200,
    send: "Hello, world!",
    listen: 8000
});
Now we use it: 
node helloworld.js
