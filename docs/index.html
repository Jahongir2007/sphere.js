Why Sphere.js

Sphere.js is used to create simple
http servers. This framework is a good choice
for beginners. Learning Sphere.js is very quick and very easy.

const sphere = require("./sphere");
const app = sphere;
app.options({
    head: 200,
    send: "Creating HTTP server",
    listen: 8000
});

Easy framework

If you want to reach your goal with short codes. So this framework is for you.

Fast API for HTTP servers

Fast and easy 3 API. Framework size is 1.32 KB.Do you like working with http servers? Then Sphere.js is a good framework for you.

Easy to learn

Sphere.js is very easy to learn. All you have to do is know what an http server is. Create a server using 3 lines of code.
